##Required Software

#LTspice XVII

Download from official site:
Available: [https://www.analog.com/en/design-center/design-tools-and-calculators/ltspice-simulator.html]
Used for schematic design and transient simulations.

#Microwind
Download from official site:
Available: [http://www.microwind.net/]
Used for layout design, rule checking, and layout simulation.

##Project Contents

--> "inverter_chain_buffer.asc" in the "Inverter_Chain_Buffer" folder
Open this file in "LTspice" to view the schematic and run transient simulations.

--> "Layout.MSK"
Open this file in "Microwind" to view the layout and run the layout simulation.
-----------------------------------------------------------------------------------------------------
                                               Steps to Run
------------------------------------------------------------------------------------------------------
1.Export all documents from the repository.
2.Open inverter_chain_buffer.asc in LTspice XVII.
3.Run the simulation to view the schematic behavior and waveforms.
4.Open Layout.MSK in Microwind.
5.Verify the layout, run DRC/LVS, and perform layout simulations.
6.Refer to inverter_chain_buffer_report.pdf for detailed documentation.
